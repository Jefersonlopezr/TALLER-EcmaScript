¿Qué es ECMAScript?

ECMAScript es el estándar creado por ECMA International para definir cómo debe funcionar el lenguaje JavaScript. Gracias a esta especificación, todos los navegadores y entornos pueden ejecutar código JavaScript de manera coherente.

JavaScript es el lenguaje basado en esta especificación. Las nuevas versiones de ECMAScript introducen mejoras de sintaxis, nuevas funcionalidades, y correcciones que hacen que el desarrollo web sea más potente y moderno.
